# Per Boba

Pagina romantica con rosa 3D che sbocchia mentre scorri.

## Avvio locale

Serve un server HTTP (i moduli ES e il modello OBJ non funzionano con `file://`):

```bash
python -m http.server 8765
```

Apri [http://localhost:8765](http://localhost:8765)

## File

- `index.html` — pagina e animazione Three.js
- `favicon.svg` — icona della tab
- `red_rose3.obj` — modello 3D rosa ([archive3d](https://archive3d.net), CC0)
